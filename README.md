# BankDemo ‚Äî Java Console Banking System

**BankDemo** is a Java console-based banking application that simulates the core operations of a bank account system. It provides a menu-driven interface for creating and managing accounts, making deposits and withdrawals, checking balances, and demonstrating account-specific behavior. The project is designed as a learning-focused demo for Object-Oriented Programming (OOP) in Java.

This application models typical bank workflows in a simplified, educational format:

- Create and manage bank accounts (account number, initial balance)

- Menu-driven console UI for straightforward interaction
